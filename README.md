# HALLOWTIDE MACRO
1. Ensure you have [Python](https://www.python.org/) installed on your system, if not then make sure you add it to PATH when installing
2. open a command window, cd into your hallowtide_macro directory and run "pip install -r requirements.txt", this will ensure you have everything needed for the program to run
3. setup your roblox windows so that your main account is on the left, and your alt is on the right
4. Follow the youtube tutorial to see exactly how to setup the macro https://youtu.be/73YsRUYn5u4?si=1fkyjSxrlUBOd8vn
