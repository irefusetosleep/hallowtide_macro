# HALLOWTIDE MACRO
1. Ensure you have [Python](https://www.python.org/) installed on your system, if not then make sure you add it to PATH when installing. Also be sure to download the program from the green "CODE" button and unzip the zip file.
2. Open your hallowtide_macro folder, click on the bar at the top that shows the path to the folder, and type "cmd" to open a command window. In that window run "pip install -r requirements.txt" and it'll install all the dependancies for the program to run
3. setup your roblox windows so that your main account is on the left, and your alt is on the right
4. Follow the youtube tutorial to see exactly how to setup the macro https://youtu.be/73YsRUYn5u4?si=1fkyjSxrlUBOd8vn
